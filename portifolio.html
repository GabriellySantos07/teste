O que significa a sigla html?

Sigla para HyperText Markup Language — Linguagem de Marcação de Hipertexto, o HTML é o componente base da web.

    Vantagens de ultilizar o html:

  • Estruturação do conteúdo
  • Integração com outras tecnologia
  • Aprendizado e adoção
  • Atributos de dados que podem ser personalizados
  • Suporte para recursos de aplicativos web
